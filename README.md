# Separator Sizing

### Version Information/updates
	- 1.0 (not available)
			written in C++
			accepted user input through the terminal 
			could only calculate one separator at a time
			needed to be restarted to calculate more inputs
			table 2.9 presented in text

			written in java 
			accepted user input through the terminal

			ran using .jar file
	- 2.1.0 (not available)
			updated GUI to include tabbed pane
			each result set displayed on a different tab 
			ran using .jar file
	- 2.1.1
			reformatted GUI for user simplicity
			can be run using either the .jar file or .exe file for those without a JRE downloaded.
